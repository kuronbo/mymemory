=========
MyMemory
=========

| メモを管理するUIプログラム。

usage
=====
1. 必要なモジュールをインストール
.. code-block::

    pip install memoplat
    pip install m_covers
    pip install checkio_taker


2. 環境変数PYTHONPATHにmymemoryのpathを追加
.. code-block::

    export PYTHONPATH='.../mymemory:$PYTHONPATH'


3. init.pyを実行
.. code-block::

    python3 init.py
